# JARVIS Web UI

Next.js 15 frontend for the JARVIS Personal AI OS.

## Deploy to Render

1. Fork / connect this repo in Render
2. **Build command:** `npm install && npm run build`
3. **Start command:** `npm start`
4. **Environment:** Node

The UI talks to the JARVIS API at `https://jarvis-api-fufo.onrender.com`.
